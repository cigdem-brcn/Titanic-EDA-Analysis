# 🚢 Titanic - Exploratory Data Analysis (EDA)

This project showcases a detailed Exploratory Data Analysis on the Titanic dataset. As a 2nd-year Computer Engineering student, I focused on data cleaning and gaining statistical insights.

## 🛠️ Tech Stack & Skills
* **Language:** Python
* **Library:** Pandas, Matplotlib
* **Key Operations:** Data Cleaning (Handling Nulls), Grouping, Aggregation, and Feature Engineering.

## 📊 Analysis Overview
- **Data Imputation:** Handled missing values in the `Cabin` column by filling them with "Unknown".
- **Statistical Insights:** Calculated survival rates based on demographics.
- **Visualizations:** Created bar charts to illustrate the survival gap between genders.

## 📂 Project Structure
- `Titanic-Analysis/`: Contains the main analysis notebook and raw datasets.
